🏔 간월산 Trail Tracer
📋 사용법
1. 카카오 개발자콘솔 → 내 앱 → 플랫폼 → Web
   → 사이트 도메인에 http://localhost:8080 추가
2. 터미널에서 python -m http.server 8080 실행
3. 브라우저에서 http://localhost:8080/ganwolsan_tracer.html
4. 앱키 입력 → 지도 로드 → 등산로 클릭클릭
5. Export → 복사 → 클로드에게 붙여넣기!
⚡ 또는 GitHub Pages 이용
저장소에 업로드 → GitHub Pages 활성화
→ 카카오 콘솔에 https://[계정].github.io 등록
🔑 카카오 개발자 콘솔 → 도메인 등록하러 가기
